# Excel_IA
## Repositório com arquivos do curso de Excel com IA da DIO

### Descrição do projeto 1 - Scarpa Invest

-> Planilha para investidores iniciantes que simula cenários de investimento em fundos imobiliários;

-> Inputs básicos:
Salários;
Rendimento da carteira;
Decisão de investimento mensal;
Tempo de investimento;
Perfil do investidor.

-> Outputs:
Sugestão de investimento mínima com base no salário;
Patrimônio acumulado e dividendos mensais obtidos com base no rendimento;
Cenários de investimento para períodos típicos;
Sugestão de composição da carteira de investimento com base no perfil de investidor.
